# 달팽이 경주 만들기

> 달팽이 여러 마리가 **각자 스레드가 되어** 동시에 결승선을 향해 달립니다. 콘솔에 진행 막대가 동시에 차오르고, 가장 먼저 도착한 달팽이가 우승! 멀티스레드의 핵심(`Thread` 여러 개 + `sleep`)을 눈으로 보면서 배웁니다. **아래 Step을 순서대로 따라가면, 마지막 Step에서 게임이 완성됩니다.**

---

## 1. 무엇을 만드나요?

달팽이 3마리가 동시에 출발합니다. 각자 무작위로 조금씩 전진하고, 화면에는 진행 상황이 막대(`=`)로 표시됩니다. 먼저 도착하면 우승!

```
달팽이1: ===>
달팽이2: ==>
달팽이3: =====>
달팽이1: ======>
달팽이3: =========>
달팽이2: =====>
...
달팽이3: =============================>
*** 우승: 달팽이3 ***
```

> 출력이 서로 섞여 나오는 게 정상입니다 — 그게 바로 **여러 스레드가 동시에 돌고 있다는 증거**예요!

---

## 2. 요구사항 정리 (기능 명세)

|번호|기능|설명|
|---|---|---|
|1|달팽이 = 스레드|달팽이 한 마리가 독립된 `Thread`다|
|2|전진|각 달팽이는 무작위로 1~3칸씩 전진한다|
|3|진행 표시|전진할 때마다 자기 위치를 막대로 출력한다|
|4|동시 출발|달팽이 여러 마리를 동시에 출발(`start()`)시킨다|
|5|우승 판정|가장 먼저 결승선에 닿은 달팽이를 우승자로 한 번만 선언한다|

---

## 3. 핵심 개념

### (1) 달팽이 하나 = 스레드 하나

움직이는 객체는 `Thread`를 상속하고 `run()` 안에서 **"전진 → 출력 → 잠깐 sleep"** 을 반복합니다.

```java
public void run() {
    while (position < FINISH) {
        position += rand.nextInt(3) + 1;  // 1~3칸 전진
        printProgress();                  // 현재 위치 출력
        Thread.sleep(300);                // 잠깐 멈춤 (속도 조절)
    }
}
```

### (2) `start()`로 동시에 출발

여러 달팽이를 만들어 각각 `start()`를 부르면, **동시에** `run()`이 돌기 시작합니다. (`run()`을 직접 부르면 순서대로 실행돼서 경주가 안 됩니다 — 꼭 `start()`!)

### (3) 우승자는 한 번만 — 공유 깃발

여러 스레드가 거의 동시에 도착할 수 있습니다. "이미 우승자가 정해졌나?"를 모두가 함께 보는 **공유 값**이 필요하고, 그 선언이 딱 한 번만 되도록 보호해야 합니다.

- 여러 스레드가 함께 읽는 값 → `volatile`
- 선언이 한 번만 되도록 → `synchronized`

## 5. Step by Step

각 Step에 **목표 / 할 일 / 힌트 / 확인 방법**이 있습니다.

---

### Step 1. 달팽이 한 마리 움직이기 (`Snail.java`)

**목표**: 달팽이 한 마리가 결승선까지 무작위로 전진하게 만든다. (스레드 첫 등장)

**할 일**

1. `Snail extends Thread`, 필드 `name`, `position`(시작 0), `FINISH`(예: 30).
2. `run()`에서 `position < FINISH`인 동안 1~3칸 전진하고 `sleep`.
3. 도착하면 "○○ 도착!" 출력.

**힌트**

```java
import java.util.Random;

public class Snail extends Thread {
    private String name;
    private int position = 0;
    private final int FINISH = 30;
    private Random rand = new Random();

    public Snail(String name) { this.name = name; }

    public void run() {
        while (position < FINISH) {
            position += rand.nextInt(3) + 1;   // 1~3칸
            System.out.println(name + " 위치: " + position);
            try { Thread.sleep(300); } catch (InterruptedException e) {}
        }
        System.out.println(name + " 결승선 도착!");
    }
}
// Main 에서 임시 테스트
Snail s = new Snail("달팽이1");
s.start();
```

**확인**: 달팽이 위치가 0→...→30까지 늘다가 "도착!"이 뜨면 성공.

---

### Step 2. 진행 막대로 보기 좋게 (`Snail.java`)

**목표**: 숫자 대신 막대(`=`)로 진행 상황을 표시한다.

**할 일**: 위치만큼 `=`를 그리고 끝에 `>`를 붙여 출력하는 메서드를 만든다.

**힌트**

```java
private void printProgress() {
    StringBuilder bar = new StringBuilder();
    for (int i = 0; i < position; i++) bar.append("=");
    bar.append(">");
    System.out.println(name + ": " + bar);
}
// run() 안의 출력 부분을 printProgress(); 로 교체
```

**확인**: `달팽이1: =====>` 처럼 막대가 점점 길어지면 성공.

---

### Step 3. 달팽이 여러 마리 동시 출발 (`Main.java`)

**목표**: 달팽이 3마리를 만들어 **동시에** 출발시킨다. (멀티스레드의 핵심 체감!)

**할 일**

1. 달팽이 3마리 생성.
2. 각각 `start()` 호출.

**힌트**

```java
public class Main {
    public static void main(String[] args) {
        Snail s1 = new Snail("달팽이1");
        Snail s2 = new Snail("달팽이2");
        Snail s3 = new Snail("달팽이3");
        s1.start();
        s2.start();
        s3.start();
    }
}
```

**확인**: 세 달팽이의 막대가 **서로 섞여서** 동시에 길어지면 성공. (출력이 뒤섞이는 게 정상! 동시에 돌고 있다는 뜻입니다.)

> 실험: `start()` 대신 `run()`을 직접 불러보세요. 한 마리가 끝나야 다음이 시작돼서 경주가 안 됩니다. "왜 `start()`여야 하는가"를 직접 확인하는 좋은 기회예요.

---

### Step 4. 우승자 가리기 (`Race.java`)

**목표**: 가장 먼저 도착한 달팽이를 우승자로 **딱 한 번** 선언하고, 나머지는 멈춘다.

**할 일**

1. `Race` 클래스에 "끝났는지" 깃발과 우승 선언 메서드를 만든다.
2. `Snail` 생성자에 `Race race`를 받아 멤버로 저장한다.
3. `run()`의 반복 조건을 `position < FINISH && !race.isOver()`로 바꾼다.
4. 결승선에 닿으면 `race.finish(name)`을 호출한다.

**힌트**

```java
// Race.java
public class Race {
    private volatile boolean over = false;          // 모두가 함께 보는 값

    public boolean isOver() { return over; }

    public synchronized void finish(String name) {  // 한 번만 선언되도록 보호
        if (!over) {
            over = true;
            System.out.println("\n*** 우승: " + name + " ***");
        }
    }
}
// Snail.java (수정)
private Race race;
public Snail(String name, Race race) { this.name = name; this.race = race; }
public void run() {
    while (position < FINISH && !race.isOver()) {
        position += rand.nextInt(3) + 1;
        printProgress();
        try { Thread.sleep(300); } catch (InterruptedException e) {}
    }
    if (position >= FINISH) race.finish(name);   // 도착한 경우만 우승 시도
}
// Main.java (수정)
Race race = new Race();
new Snail("달팽이1", race).start();
new Snail("달팽이2", race).start();
new Snail("달팽이3", race).start();
```

**확인**: 한 마리가 도착하면 "*** 우승: ○○ ***"이 **한 번만** 뜨고, 나머지는 곧 멈추면 성공.

---

### Step 5. 마무리 다듬기 (완성!)

**점검 항목**

- [ ] 달팽이마다 속도가 조금씩 달라 매번 결과가 바뀌는가? (`Random` 덕분)
- [ ] 우승 선언이 **딱 한 번**만 나오는가? (`synchronized` 확인)
- [ ] 우승이 정해진 뒤 나머지 달팽이가 멈추는가? (`race.isOver()` 확인)
- [ ] 막대가 결승선을 넘어 너무 길어지지 않는가? (필요하면 `if (position > FINISH) position = FINISH;`)

여기까지 통과하면 **달팽이 경주 완성입니다!** 🎉

## 8. (선택) 도전 과제

1. **마리 수 선택**: 시작 시 달팽이 몇 마리로 경주할지 입력받기
2. **베팅**: 출발 전에 "몇 번이 이길까?"를 맞히게 하고 적중 시 축하 메시지
3. **순위 매기기**: 우승뿐 아니라 도착 순서대로 1·2·3등 전부 표시 (`join()` 활용)
4. **화면 깔끔하게**: 매 틱마다 화면을 지우고 전체 트랙을 다시 그려 진짜 경주처럼 보이게 하기
5. **트랙 길이/속도 옵션**: 결승선 거리나 `sleep` 시간을 난이도로 선택
