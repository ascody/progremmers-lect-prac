# 자바 스트림 기초부터 익히기 (for문을 파이프라인으로 바꾸기)

> 스트림(Stream)을, **`for`문 + `if`문으로 짜던 걸 `만들기 → filter/map → 모으기` 파이프라인으로 바꿔가며** 익혀요. 람다를 실제로 써먹는 무대예요.
> 상품 목록을 거르고(filter), 바꾸고(map), 모으고(collect), 통계 내는 흐름을 한 줄 한 줄 쌓아 올려요.
> **아래 Step을 순서대로 따라가면, 마지막 Step에서 스트림을 자유롭게 쓰는 프로그램이 완성됩니다.**
>
> 💡 각 Step의 **힌트는 접혀 있어요.** "이 `for`문을 스트림으로 어떻게 옮기지?"를 먼저 고민하고, 막히면 펼치세요.

---

## 0. 먼저 알아둘 점

- 스트림은 **컬렉션(리스트 등)의 데이터를 흐름처럼 처리**하는 도구예요. `filter`, `map` 같은 연산에 **람다**를 넘겨서 동작을 정해요. (람다를 안다고 가정해요.)
- 스트림은 항상 **세 단계**예요: ① 스트림 **만들기**(`list.stream()`) → ② **중간 연산**(`filter`/`map`/`sorted`… 여러 개 가능) → ③ **종료 연산**(`forEach`/`collect`/`count`/`sum` 하나). 종료 연산이 와야 비로소 실행돼요.
- `List<String>`, `Stream<Product>`처럼 **이미 만들어진 제네릭을 구체 타입으로 쓰는 것**만 다뤄요. 직접 만드는 `<T>` 제네릭은 안 해요.
- 합계·평균 같은 숫자 통계는 `mapToInt(...)`로 **숫자 스트림(IntStream)** 으로 바꾼 뒤 `sum()`/`average()`를 써요.

## 3. 핵심 개념

### (1) 스트림의 3단계 (파이프라인)
```
products.stream()                 ① 만들기 (컬렉션 → 스트림)
        .filter(p -> ...)         ② 중간 연산 (스트림을 돌려줌 → 계속 이어붙임)
        .map(p -> ...)            ② 중간 연산 (여러 개 가능)
        .collect(...);            ③ 종료 연산 (결과를 냄 → 여기서 끝)
```
- **중간 연산**(`filter`, `map`, `sorted`…)은 **스트림을 돌려줘서** 점(`.`)으로 계속 이어 붙일 수 있어요.
- **종료 연산**(`forEach`, `collect`, `count`, `sum`…)은 **결과**(리스트·숫자 등)를 내고 끝나요. **종료 연산이 와야 실제로 실행**돼요.

### (2) for문 vs 스트림 (같은 일, 다른 모양)
```java
// for문: "어떻게(how)" 도는지 일일이 적는다
List<String> result = new ArrayList<>();
for (Product p : products) {
    if (p.getPrice() >= 1000) {     // 거르고
        result.add(p.getName());    // 바꿔서 담는다
    }
}

// 스트림: "무엇을(what)" 하는지만 적는다
List<String> result = products.stream()
        .filter(p -> p.getPrice() >= 1000)   // 거르고
        .map(p -> p.getName())               // 바꿔서
        .collect(Collectors.toList());       // 담는다
```

### (3) 자주 쓰는 연산과 람다의 종류
| 연산 | 하는 일 | 넘기는 람다 |
|------|--------|-----------|
| `filter` | 조건에 맞는 것만 남김 | 조건(`p -> p.getPrice() >= 1000`) → true/false |
| `map` | 다른 값으로 변환(1:1) | 변환(`p -> p.getName()`) |
| `flatMap` | 변환 결과가 리스트일 때 평탄화 | 스트림 반환(`o -> o.getItems().stream()`) |
| `forEach` | 하나씩 처리(출력 등) | 처리(`p -> System.out.println(p)`) |
| `sorted` | 정렬 | 비교(`(a, b) -> a.getPrice() - b.getPrice()`) |
| `collect` | 리스트 등으로 모음 | (보통 `Collectors.toList()`) |
| `count` | 개수 세기 | 없음 |

```
[기억법] 스트림 = "stream()으로 열고 → filter/map으로 거르고 바꾸고 → collect/forEach로 닫기"
```

## 5. Step by Step

### Step 1. 스트림 만들고 전체 출력하기 (`Product.java`, `Main.java`)

**목표**: 리스트로 스트림을 만들고 `forEach`로 하나씩 출력해요.

**할 일**
1. `Product`(name, price)를 만들고, 상품 5개가 든 `List<Product>`를 준비해요.
2. `products.stream().forEach(...)`로 "이름 (가격원)"을 출력해요.

<details>
<summary>💡 힌트 보기</summary>

```java
// Product.java
class Product {
    private String name;
    private int price;
    public Product(String name, int price) { this.name = name; this.price = price; }
    public String getName()  { return name; }
    public int    getPrice() { return price; }
}
```
```java
// Main.java 안에서
List<Product> products = new ArrayList<>(Arrays.asList(
    new Product("연필", 500), new Product("공책", 1200), new Product("지우개", 300),
    new Product("필통", 3000), new Product("볼펜", 800)
));

products.stream()
        .forEach(p -> System.out.println(p.getName() + " (" + p.getPrice() + "원)"));
```

`stream()`으로 흐름을 열고, `forEach`(종료 연산)로 하나씩 처리해요. `forEach`에 넘기는 건 "하나를 받아 처리하는" 람다예요.

</details>

**확인**: 상품 5개가 모두 "이름 (가격원)" 형태로 출력되면 통과예요.

### Step 2. filter — 조건에 맞는 것만 (`Main.java`)

**목표**: `1000원 이상` 상품만 골라 출력해요. (`for` + `if` → `filter`)

**할 일**
1. `stream()` 뒤에 `.filter(...)`를 붙여 가격이 1000 이상인 것만 남겨요.
2. 그 뒤에 `.forEach(...)`로 출력해요.

<details>
<summary>💡 힌트 보기</summary>

```java
products.stream()
        .filter(p -> p.getPrice() >= 1000)     // 조건이 true인 것만 통과
        .forEach(p -> System.out.println(p.getName() + " (" + p.getPrice() + "원)"));
```

`filter`에 넘기는 람다는 **true/false를 돌려주는 조건**이에요. true인 것만 다음 단계로 흘러가요. (`if`문이 한 줄로 들어간 셈이에요.)

</details>

**확인**: 공책·필통만 출력되면 성공이에요.

---

### Step 3. map — 다른 값으로 변환 (`Main.java`)

**목표**: 상품(`Product`)을 **이름(`String`)으로 변환**해 출력해요.

**할 일**
1. `stream()` 뒤에 `.map(p -> p.getName())`을 붙여 상품을 이름으로 바꿔요.
2. `.forEach(...)`로 이름만 출력해요.

<details>
<summary>💡 힌트 보기</summary>

```java
products.stream()
        .map(p -> p.getName())                 // Product -> String 으로 변환
        .forEach(name -> System.out.println(name));
```

`map`은 **각 요소를 다른 값으로 바꿔요**. 여기선 `Product`가 흐르던 스트림이 `map` 이후엔 `String`(이름)이 흐르는 스트림으로 바뀌어요.

</details>

**확인**: 상품 이름 5개만 한 줄씩 출력되면 성공이에요.

### Step 4. map vs flatMap — 평탄화 (`Order.java`, `Main.java`)

**목표**: 변환 결과가 **또 리스트**일 때, `map`은 중첩되고 `flatMap`은 한 줄로 펴진다는 걸 직접 봐요.

이번엔 "주문 하나가 **상품 이름 리스트**를 담는" 데이터를 써요. 즉 *리스트 안에 리스트*예요.

**할 일**
1. `Order`(id, `List<String> items`)를 만들고, 주문 2건을 준비해요.
2. `map(o -> o.getItems())` 결과를 출력해, **리스트의 리스트(중첩)** 가 나오는 걸 확인해요.
3. `flatMap(o -> o.getItems().stream())` 결과를 출력해, **하나로 펴진(평탄)** 리스트가 나오는 걸 확인해요.

<details>
<summary>💡 힌트 보기</summary>

```java
// Order.java
class Order {
    private int id;
    private java.util.List<String> items;
    public Order(int id, java.util.List<String> items) { this.id = id; this.items = items; }
    public java.util.List<String> getItems() { return items; }
}
```
```java
// Main.java 안에서
List<Order> orders = Arrays.asList(
    new Order(1, Arrays.asList("연필", "공책")),
    new Order(2, Arrays.asList("필통", "볼펜", "공책"))
);

// map: 주문 -> 상품'리스트'  => 리스트의 리스트 (중첩!)
List<List<String>> byMap = orders.stream()
        .map(o -> o.getItems())
        .collect(Collectors.toList());
System.out.println("map     : " + byMap);     // [[연필, 공책], [필통, 볼펜, 공책]]

// flatMap: 각 리스트를 stream()으로 풀어 하나로 합침 => 평탄!
List<String> byFlatMap = orders.stream()
        .flatMap(o -> o.getItems().stream())
        .collect(Collectors.toList());
System.out.println("flatMap : " + byFlatMap); // [연필, 공책, 필통, 볼펜, 공책]
```

차이는 한 가지예요. **`map`에 넘긴 람다는 "리스트"를 돌려주고**, **`flatMap`에 넘긴 람다는 "스트림"을 돌려줘요**(그래서 `.stream()`을 붙여요). 그러면 flatMap이 그 작은 스트림들을 하나로 이어 붙여 평평하게 만들어요.

```
map     :  주문 ──▶ [상품,상품]        => Stream< List<String> >   (껍질이 남음=중첩)
flatMap :  주문 ──▶ 상품,상품  (풀어서)  => Stream< String >         (껍질을 벗김=평탄)
```

</details>

**확인**: `map`은 `[[연필, 공책], [필통, 볼펜, 공책]]`(중첩), `flatMap`은 `[연필, 공책, 필통, 볼펜, 공책]`(평탄)이 나오면 성공이에요.

### Step 5. filter + map + collect 파이프라인 (`Main.java`)

**목표**: "1000원 이상 상품의 **이름 리스트**"를 한 번에 만들어요. (거르고 → 바꾸고 → 모으고)

**할 일**
1. `.filter(...)`로 1000원 이상만 남기고,
2. `.map(...)`으로 이름으로 바꾸고,
3. `.collect(Collectors.toList())`로 **새 리스트**에 모아요.
4. 그 리스트를 출력해요.

<details>
<summary>💡 힌트 보기</summary>

```java
import java.util.stream.Collectors;

List<String> expensiveNames = products.stream()
        .filter(p -> p.getPrice() >= 1000)     // 거르고
        .map(p -> p.getName())                 // 바꾸고
        .collect(Collectors.toList());         // 모은다(종료 연산)

System.out.println(expensiveNames);            // [공책, 필통]
```

`collect(Collectors.toList())`가 흐르던 값들을 **하나의 리스트로 모아** 돌려줘요. 중간 연산(filter, map)을 점으로 죽 이어 붙이다가 마지막에 종료 연산으로 닫는 게 핵심이에요.

</details>

**확인**: 출력이 `[공책, 필통]`이면 성공이에요.

### Step 6. 통계 — count / sum / average / sorted (`Main.java`)

**목표**: 개수·합계·평균을 구하고, 가격순으로 정렬해요.

**할 일**
1. `count()`로 1000원 이상 상품 **개수**를 구해요.
2. `mapToInt(p -> p.getPrice()).sum()`으로 **가격 합계**, `.average().getAsDouble()`로 **평균**을 구해요.
3. `sorted((a, b) -> a.getPrice() - b.getPrice())`로 **가격 오름차순** 정렬 후 이름 리스트를 만들어요.

<details>
<summary>💡 힌트 보기</summary>

```java
long count = products.stream()
        .filter(p -> p.getPrice() >= 1000)
        .count();                                       // 개수(종료 연산)

int sum = products.stream()
        .mapToInt(p -> p.getPrice())                    // 숫자 스트림으로
        .sum();                                         // 합계

double avg = products.stream()
        .mapToInt(p -> p.getPrice())
        .average().getAsDouble();                       // 평균

List<String> byPrice = products.stream()
        .sorted((a, b) -> a.getPrice() - b.getPrice())  // 가격 오름차순
        .map(p -> p.getName())
        .collect(Collectors.toList());
```

합계·평균은 `Product` 스트림 그대로는 못 구해요. `mapToInt`로 **가격(int)만 흐르는 숫자 스트림**으로 바꾼 뒤 `sum()`/`average()`를 써요. (`average()`는 값이 없을 수도 있어 `getAsDouble()`로 꺼내요.)

</details>

**확인**: 개수 `2`, 합계 `5800`, 평균 `1160.0`, 정렬 `[지우개, 연필, 볼펜, 공책, 필통]`이면 성공이에요.
