# 빙고 게임 만들기 (2차원 배열 활용)

> 콘솔에서 동작하는 **빙고 게임**을 만들면서 `2차원 배열`, `반복문`, `메서드`, `난수(Random)`, `조건문`을 종합적으로 연습합니다. 컴퓨터와 번갈아 숫자를 부르고, 먼저 정해진 줄 수만큼 빙고를 완성하면 승리합니다. **아래 Step을 순서대로 따라가면, 마지막 Step에서 게임이 완성됩니다.**

---

## 1. 무엇을 만드나요?

콘솔(터미널)에서 컴퓨터와 1:1로 겨루는 5×5 빙고 게임입니다.

- 나와 컴퓨터가 각자 1~25 숫자가 **무작위로 섞인** 빙고판을 가진다.
- 내 차례에 숫자 하나를 부르면, **나와 컴퓨터 두 판 모두**에서 그 숫자가 지워진다.
- 이어서 컴퓨터가 무작위로 숫자 하나를 부른다. (역시 두 판 모두 지워짐)
- 가로·세로·대각선 한 줄이 모두 지워지면 **빙고 1줄**.
- 먼저 목표 줄 수(기본 3줄)를 완성하는 쪽이 **승리**.

실행 화면 예시:

```
===== 내 빙고판 =====
[ 7] [13] [ 1] [22] [ 9]
[ 4] [ ★] [18] [ ★] [ 2]
[25] [11] [ ★] [ 6] [16]
[ 3] [20] [ ★] [14] [ ★]
[ 8] [17] [21] [10] [ 5]

부를 숫자 입력 (1~25) > 7

▶ 내가 부른 숫자: 7
▶ 컴퓨터가 부른 숫자: 18

현재 빙고 줄  → 나: 1줄,  컴퓨터: 0줄
```

> `★` 은 이미 불려서 지워진(마킹된) 칸입니다.

---

## 2. 요구사항 정리 (기능 명세)

|번호|기능|설명|
|---|---|---|
|1|빙고판 만들기|1~25 숫자를 무작위로 섞어 5×5 판에 채운다. (나, 컴퓨터 각각)|
|2|빙고판 출력|판을 보기 좋게 출력한다. 지워진 칸은 `★` 로 표시한다.|
|3|숫자 부르기|내 차례엔 직접 입력, 컴퓨터 차례엔 무작위로 고른다. 이미 부른 숫자는 다시 못 부른다.|
|4|마킹|부른 숫자를 **두 판 모두**에서 찾아 지운다.|
|5|빙고 줄 세기|가로 5줄 + 세로 5줄 + 대각선 2줄 중 완성된 줄 수를 센다.|
|6|승패 판정|먼저 목표 줄 수에 도달한 쪽이 승리. (동시 도달이면 무승부)|

## 3. 핵심 개념: 데이터를 어떻게 저장할까?

빙고판은 **5행 5열의 격자**입니다. 이걸 그대로 `2차원 배열`로 표현합니다.

```
int[][] board = new int[5][5];   // 칸마다 숫자(1~25)
boolean[][] marked = new boolean[5][5];   // 칸마다 지워졌는지(true/false)
```

- `board[r][c]` → r행 c열에 적힌 **숫자**
- `marked[r][c]` → 그 칸이 **지워졌는지** 여부
- 이미 부른 숫자를 또 부르지 못하게 하려면 `boolean[] called = new boolean[26]` (1~25) 로 기록

> 이 과제의 핵심은 **2차원 배열을 이중 for문으로 순회**하는 것입니다. (채우기 / 출력 / 마킹 / 줄 세기 모두 이중 for문)

### 빙고 줄을 세는 법

```
가로 한 줄: 같은 r 에서 c=0~4 가 모두 marked 인가?
세로 한 줄: 같은 c 에서 r=0~4 가 모두 marked 인가?
대각선 ＼ : marked[0][0], [1][1], [2][2], [3][3], [4][4] 가 모두?
대각선 ／ : marked[0][4], [1][3], [2][2], [3][1], [4][0] 가 모두?
```

최대 12줄(가로5+세로5+대각2)까지 나올 수 있습니다.

## 5. Step by Step

각 Step에는 **목표 / 할 일 / 힌트 / 확인 방법**이 있습니다. 한 Step을 끝내고 실행이 잘 되는지 확인한 뒤 다음 Step으로 넘어가세요.

---

### Step 1. 시작 화면 띄우기 (`Start.java`, `BingoGame.java`)

**목표**: `BingoGame` 객체를 만들고 `play()`를 호출하면 시작 인사말이 뜨게 한다.

**할 일**

- `BingoGame` 클래스에 `play()` 메서드를 만들고 인사말만 출력한다.
- `Start`의 `main`에서 `BingoGame` 객체를 만들고 `play()`를 호출한다.

**힌트**

```java
// BingoGame.java
public class BingoGame {
    public void play() {
        System.out.println("===== 빙고 게임 =====");
        System.out.println("컴퓨터와 번갈아 숫자를 불러 빙고를 완성하세요!");
    }
}

// Start.java
public class Start {
    public static void main(String[] args) {
        BingoGame game = new BingoGame();
        game.play();
    }
}
```

**확인**: 실행하면 인사말이 출력되면 성공.

---

### Step 2. 빙고판 만들기 (`makeBoard`)

**목표**: 1~25 숫자를 무작위로 섞어 5×5 배열에 채운다.

**할 일**

1. 필드로 판 크기 상수와 배열들을 선언한다.
2. 1~25를 리스트에 담고 `Collections.shuffle`로 섞는다.
3. 섞은 숫자를 이중 for문으로 `board[r][c]`에 하나씩 채운다.

**힌트**

```java
static final int SIZE = 5;        // 판 크기 5x5
static final int MAX = 25;        // 숫자 1~25

void makeBoard(int[][] board) {
    List<Integer> nums = new ArrayList<>();
    for (int i = 1; i <= MAX; i++) nums.add(i);
    Collections.shuffle(nums);    // 무작위로 섞기

    int idx = 0;
    for (int r = 0; r < SIZE; r++)
        for (int c = 0; c < SIZE; c++)
            board[r][c] = nums.get(idx++);
}
```

**확인**: 다음 Step에서 출력해보면 1~25가 겹치지 않고 무작위로 들어가 있으면 성공.

---

### Step 3. 빙고판 출력하기 (`printBoard`)

**목표**: 판을 격자 모양으로 보기 좋게 출력한다. 지워진 칸은 `★`로 표시.

**할 일**

- 이중 for문으로 한 칸씩 출력한다.
- `marked[r][c]`가 `true`면 `★`, 아니면 숫자를 출력한다.
- 두 자리로 칸을 맞추면 깔끔하다. (`%2d`)

**힌트**

```java
void printBoard(int[][] board, boolean[][] marked) {
    for (int r = 0; r < SIZE; r++) {
        for (int c = 0; c < SIZE; c++) {
            if (marked[r][c]) System.out.print("[ ★] ");
            else              System.out.printf("[%2d] ", board[r][c]);
        }
        System.out.println();   // 한 줄 끝나면 줄바꿈
    }
}
```

**확인**: 5×5 격자가 가지런히 출력되면 성공.

---

### Step 4. 숫자 마킹하기 (`mark`)

**목표**: 부른 숫자를 판에서 찾아 `marked`를 `true`로 만든다.

**할 일**

- 이중 for문으로 판 전체를 돌면서, `board[r][c]`가 부른 숫자와 같으면 그 칸을 마킹한다.

**힌트**

```java
void mark(int[][] board, boolean[][] marked, int num) {
    for (int r = 0; r < SIZE; r++)
        for (int c = 0; c < SIZE; c++)
            if (board[r][c] == num)
                marked[r][c] = true;
}
```

**확인**: 마킹 후 출력해보면 해당 숫자 칸이 `★`로 바뀌면 성공.

---

### Step 5. 빙고 줄 세기 (`countBingo`)

**목표**: 완성된 줄(가로/세로/대각선)의 개수를 센다.

**할 일**

1. 가로 5줄을 검사한다. (각 행이 전부 마킹됐는지)
2. 세로 5줄을 검사한다.
3. 대각선 2줄을 검사한다.
4. 완성된 줄 수를 합쳐 반환한다.

**힌트**

```java
int countBingo(boolean[][] marked) {
    int count = 0;

    // 가로
    for (int r = 0; r < SIZE; r++) {
        boolean all = true;
        for (int c = 0; c < SIZE; c++) if (!marked[r][c]) all = false;
        if (all) count++;
    }
    // 세로
    for (int c = 0; c < SIZE; c++) {
        boolean all = true;
        for (int r = 0; r < SIZE; r++) if (!marked[r][c]) all = false;
        if (all) count++;
    }
    // 대각선 ＼
    boolean d1 = true;
    for (int i = 0; i < SIZE; i++) if (!marked[i][i]) d1 = false;
    if (d1) count++;
    // 대각선 ／
    boolean d2 = true;
    for (int i = 0; i < SIZE; i++) if (!marked[i][SIZE - 1 - i]) d2 = false;
    if (d2) count++;

    return count;
}
```

**확인**: 한 줄을 일부러 다 마킹시켜 보고 `countBingo`가 1을 반환하면 성공.

---

### Step 6. 숫자 부르기 (`playerPick`, `computerPick`)

**목표**: 내 차례엔 입력으로, 컴퓨터 차례엔 난수로 숫자를 고른다. 이미 부른 숫자는 못 고르게 막는다.

**할 일**

1. `boolean[] called = new boolean[MAX + 1]` 로 이미 부른 숫자를 기록한다.
2. `playerPick`: 1~25 범위인지, 이미 부른 숫자가 아닌지 검사하며 다시 입력받는다.
3. `computerPick`: 아직 안 부른 숫자가 나올 때까지 `Random`으로 뽑는다.

**힌트**

```java
boolean[] called = new boolean[MAX + 1];   // 1~25
Scanner sc = new Scanner(System.in);
Random rand = new Random();

int playerPick() {
    while (true) {
        System.out.print("부를 숫자 입력 (1~25) > ");
        int num;
        try {
            num = Integer.parseInt(sc.nextLine().trim());
        } catch (NumberFormatException e) {
            System.out.println("숫자만 입력하세요."); continue;
        }
        if (num < 1 || num > MAX)      System.out.println("1~25 사이로 입력하세요.");
        else if (called[num])          System.out.println("이미 부른 숫자입니다.");
        else                           return num;
    }
}

int computerPick() {
    int num;
    do { num = rand.nextInt(MAX) + 1; } while (called[num]);
    return num;
}
```

**확인**: 잘못된 값이나 이미 부른 숫자를 넣으면 다시 묻고, 올바른 숫자만 통과하면 성공.

---

### Step 7. 게임 진행 & 승패 판정 (`play`)

**목표**: 두 판을 만들고, 번갈아 숫자를 부르며, 누군가 목표 줄 수에 도달하면 끝낸다.

**할 일**

1. `play()`에서 두 판을 `makeBoard`로 만든다.
2. `while(true)` 안에서 내 판을 출력 → 내가 숫자 부르기 → 두 판 마킹.
3. 이어서 컴퓨터가 숫자 부르기 → 두 판 마킹.
4. 매 호출 후 양쪽 빙고 줄 수를 비교해 **목표 도달 시 승패 출력 후 종료**.

**힌트**

```java
static final int TARGET = 3;   // 3줄 먼저 만들면 승리

public void play() {
    makeBoard(playerBoard);
    makeBoard(computerBoard);
    System.out.println("먼저 " + TARGET + "줄을 완성하면 승리!");

    while (true) {
        System.out.println("\n===== 내 빙고판 =====");
        printBoard(playerBoard, playerMarked);

        // 내 차례
        int num = playerPick();
        callNumber(num, "내가");
        if (checkWin()) break;

        // 컴퓨터 차례
        int cNum = computerPick();
        callNumber(cNum, "컴퓨터가");
        if (checkWin()) break;

        System.out.println("\n현재 빙고 줄  → 나: " + countBingo(playerMarked)
                         + "줄,  컴퓨터: " + countBingo(computerMarked) + "줄");
    }
}
```

> `callNumber(숫자, 누구)` 와 `checkWin()` 은 중복을 줄이기 위한 도우미 메서드입니다. (예제코드 참고)

**확인**: 게임이 끝까지 진행되고, 한쪽이 3줄을 만들면 승패가 출력되며 종료되면 성공.

- [ ] 숫자 입력란에 글자를 넣어도 프로그램이 죽지 않는가?

## 7. (선택) 도전 과제

여유가 있는 학생은 아래 중 골라서 확장해 보세요.

1. **빙고 외치기**: 줄이 완성될 때마다 "빙고!"를 한 번씩만 출력 (이미 외친 줄은 다시 안 외치기)
2. **줄 수 선택**: 시작할 때 목표 줄 수(3/4/5)를 입력받기
3. **판 크기 변경**: `SIZE`만 바꿔도 3×3, 7×7로 동작하도록 (`MAX = SIZE*SIZE`)
4. **2인용**: 컴퓨터 대신 사람 2명이 번갈아 입력하기
5. **숫자 직접 채우기**: 무작위 대신 사용자가 원하는 위치에 숫자를 직접 배치
6. **기록 저장**: 승패 결과를 파일에 누적 기록 (전적: 3승 1패)
