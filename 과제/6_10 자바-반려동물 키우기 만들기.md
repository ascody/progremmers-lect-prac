# 반려동물 키우기 만들기 (객체지향 · 생성자 · 입문)

> 반려동물 하나를 입양해서 먹이를 주고 놀아주는 콘솔 게임입니다. 객체를 **하나만** 다루고, 메서드는 그 객체의 값(포만감·행복)만 바꾸기 때문에 객체지향이 처음인 학생도 부담 없이 **클래스·객체·생성자·캡슐화**를 익힐 수 있어요. **아래 Step을 순서대로 따라가면, 마지막 Step에서 게임이 완성됩니다.**

---

## 1. 무엇을 만드나요?

이름을 지어 반려동물을 입양하고, 먹이주기/놀아주기로 포만감과 행복을 올립니다.

```
반려동물의 이름을 지어주세요: 콩이
콩이이(가) 태어났어요!
[콩이] 포만감: 50 / 행복: 50

무엇을 할까요? [1]먹이주기 [2]놀아주기 [3]상태보기 [4]종료
> 1
콩이에게 먹이를 줬어요! 냠냠
[콩이] 포만감: 70 / 행복: 55
> 2
콩이와(과) 신나게 놀았어요!
[콩이] 포만감: 60 / 행복: 75
```

---

## 2. 학습 목표 (이 게임으로 배우는 것)

|개념|어디서 배우나|
|---|---|
|클래스와 객체|반려동물 설계도(클래스)로 내 펫(객체)을 하나 만든다|
|생성자|펫을 `new` 할 때 이름을 정하고 기본 상태를 채운다|
|`this` 키워드|생성자에서 필드와 매개변수를 구분한다|
|캡슐화|포만감·행복은 `private`, 변화는 메서드로만|
|메서드로 상태 바꾸기|`feed()`, `play()`가 객체의 값을 바꾼다|

> RPG와 달리 **객체가 하나뿐**이고, 객체끼리 주고받는 복잡한 동작이 없어 훨씬 쉽습니다.

---

## 3. 핵심 개념

### (1) 클래스 = 설계도, 객체 = 내 펫

`Pet`이라는 설계도(클래스)를 만들면, 그걸로 "콩이"라는 실제 펫(객체)을 찍어냅니다.

### (2) 생성자 = 펫을 만들 때 기본 상태 채우기

```java
public Pet(String name) {
    this.name = name;     // 이름은 입양할 때 정함
    this.fullness = 50;   // 포만감 기본값
    this.happiness = 50;  // 행복 기본값
}
// 사용
Pet pet = new Pet("콩이");   // 생성자가 호출되어 콩이가 태어남
```

### (3) `this` — 이 펫의 필드

매개변수 `name`과 필드 `name`의 이름이 같을 때, `this.name`은 "이 객체의 필드"를 뜻합니다.

### (4) 캡슐화 — 값은 숨기고 메서드로만 바꾸기

포만감·행복을 아무 데서나 직접 바꾸지 못하게 `private`으로 숨기고, `feed()`·`play()` 같은 메서드를 통해서만 바꾸도록 합니다.

## 5. Step by Step

각 Step에 **목표 / 할 일 / 힌트 / 확인 방법**이 있습니다.

---

### Step 1. 반려동물 설계도와 생성자 (`Pet.java`)

**목표**: 이름·포만감·행복을 가진 펫 클래스를 만들고, 생성자로 펫 하나를 만들어 상태를 본다.

**할 일**

1. 필드 `name`, `fullness`(포만감), `happiness`(행복) — 우선 `private`.
2. 이름을 받는 생성자 작성. 포만감·행복은 기본값 50으로.
3. 상태를 출력하는 `showStatus()` 메서드.
4. `Main`에서 펫 하나 만들어 상태 출력.

**힌트**

```java
public class Pet {
    private String name;
    private int fullness;
    private int happiness;

    public Pet(String name) {
        this.name = name;
        this.fullness = 50;
        this.happiness = 50;
    }

    public void showStatus() {
        System.out.println("[" + name + "] 포만감: " + fullness + " / 행복: " + happiness);
    }
}
// Main
Pet pet = new Pet("콩이");
pet.showStatus();   // [콩이] 포만감: 50 / 행복: 50
```

**확인**: "[콩이] 포만감: 50 / 행복: 50"이 나오면 성공. 생성자가 호출된 것!

---

### Step 2. 먹이주기 메서드 (`Pet.java`)

**목표**: 먹이를 주면 포만감이 오르는 메서드를 만든다.

**할 일**

1. `feed()` 메서드: 포만감을 20 올리되 **최대 100**을 넘지 않게.
2. (덤) 먹으면 기분도 조금 좋아지게 행복 +5.
3. 메시지를 출력.

**힌트**

```java
public void feed() {
    fullness += 20;
    if (fullness > 100) fullness = 100;   // 100 초과 방지
    happiness += 5;
    if (happiness > 100) happiness = 100;
    System.out.println(name + "에게 먹이를 줬어요! 냠냠");
}
// Main 에서 확인
pet.feed();
pet.showStatus();   // 포만감 70, 행복 55
```

**확인**: `feed()` 후 포만감이 올라가면 성공. 여러 번 줘도 100을 안 넘는지 확인!

---

### Step 3. 놀아주기 메서드 (`Pet.java`)

**목표**: 놀아주면 행복이 오르지만 배는 조금 고파지는 메서드를 만든다.

**할 일**

1. `play()` 메서드: 행복 +20 (최대 100), 포만감 -10 (**최소 0**).
2. 메시지를 출력.

**힌트**

```java
public void play() {
    happiness += 20;
    if (happiness > 100) happiness = 100;
    fullness -= 10;
    if (fullness < 0) fullness = 0;        // 0 미만 방지
    System.out.println(name + "와(과) 신나게 놀았어요!");
}
```

**확인**: `play()` 후 행복은 오르고 포만감은 내려가면 성공.

---

### Step 4. 메뉴로 직접 돌보기 (`Main.java`)

**목표**: 먹이주기/놀아주기/상태보기를 메뉴에서 골라 반복할 수 있게 한다.

**할 일**

1. 이름을 입력받아 펫을 입양(`new Pet(name)`).
2. `while`로 메뉴를 반복 출력.
3. 번호에 따라 `feed()`/`play()`/`showStatus()` 호출, 4번이면 종료.

**힌트**

```java
import java.util.Scanner;

Scanner sc = new Scanner(System.in);
System.out.print("반려동물의 이름을 지어주세요: ");
String name = sc.nextLine();
Pet pet = new Pet(name);
pet.showStatus();

while (true) {
    System.out.println("\n무엇을 할까요? [1]먹이주기 [2]놀아주기 [3]상태보기 [4]종료");
    System.out.print("> ");
    int menu = Integer.parseInt(sc.nextLine());  // nextLine 으로 받아 숫자 변환(입력 꼬임 방지)

    if (menu == 1)      { pet.feed();  pet.showStatus(); }
    else if (menu == 2) { pet.play();  pet.showStatus(); }
    else if (menu == 3) { pet.showStatus(); }
    else if (menu == 4) { System.out.println("안녕!"); break; }
    else                { System.out.println("1~4 중에 골라주세요."); }
}
```

> **팁**: 메뉴 번호를 `sc.nextInt()` 대신 `Integer.parseInt(sc.nextLine())`로 받으면, 숫자와 줄바꿈이 꼬여 입력이 밀리는 흔한 버그를 피할 수 있어요.

**확인**: 메뉴를 골라가며 펫의 상태가 바뀌고, 4번으로 종료되면 성공.

## 8. (선택) 도전 과제 — 다음 단계로

1. **잠자기**: `sleep()` 메서드로 포만감·행복을 회복시키기
2. **기분 표시**: 행복 수치에 따라 "행복해요 / 그저그래요 / 시무룩"을 보여주기
3. **시간 경과**: 행동할 때마다 포만감이 조금씩 자동으로 줄게 해서 계속 돌봐야 하게 만들기
4. **생성자 오버로딩**: 종류까지 정하는 생성자 `Pet(String name, String type)` 추가하기
5. **여러 마리**: 펫을 배열로 여러 마리 키우기 → 그 다음 단계가 바로 **RPG 전투(여러 객체 다루기)** 입니다!
